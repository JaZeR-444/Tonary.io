# Glossary — Tonary.io

> Canonical terms. The fuller product/terminology glossary lives in `../project/terminology.md`;
> this memory copy is the quick, durable reference. Banned-as-branding terms are marked ⛔.

## Brand & product
- **Tonary** — the official app/product name.
- **Tonary.io** — the project / web identity.
- **TONARY.wav** — hero/campaign styling only; never the app/package/store name.
- **Sound-design intelligence layer** — Tonary's product category (not "database," not "chatbot").

## Modules
- **Scout** — AI recommendations; plugin/preset/chain matching.
- **Vault** — curated plugin, preset, chain & parameter records.
- **Review** — source review and evidence-backed notes.
- **Flow** — workflow recipes, setup paths, sound-design playbooks.
- **Briefs** — fast explanations, comparisons, and next-step suggestions.

## Data record types (schemas in `../schemas/`)
- **Plugin Record** — a plugin and its metadata/parameters.
- **Preset Record** — a preset/patch snapshot for a plugin.
- **Workflow Recipe** — a Flow step-by-step recipe with a plugin chain.
- **Sound Design Note** — an evidence-backed note (Review/Briefs).
- **Source Reference** — a citation; every record's `sources[]` points here.
- **Comparison Brief** — an "X vs Y" recommendation (Briefs).
- **Learning Path** — an ordered grouping of notes/recipes.
- **User Saved Item** — a user-local saved reference (User Session).

## Design tokens (see `../brand/`)
- **outline-control** `#5a666c` — derived neutral for control borders / chart axes / focus boundaries.
- **Color law** — amber=action, cyan=focus/system, green=affirmative, violet=creative/exploratory, red=exceptional/danger.

## Tech
- **Riverpod / go_router / Drift / freezed** — recommended Flutter stack (see `../architecture/`).
- **Seed pipeline** — build-time process turning curated records into bundled JSON → local DB.

## Third-party context (FL Studio) — ⛔ never Tonary branding
- **FL Studio** — Image-Line's DAW; the platform Tonary supports.
- ⛔ **Browser**, ⛔ **Plugin Database**, ⛔ **Channel Rack**, ⛔ **Mixer**, ⛔ **Playlist** — FL Studio UI terms; do not use as Tonary module/product names.
- **Image-Line** — FL Studio's vendor; Tonary is unaffiliated.
